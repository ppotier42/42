# ft_irc

C'est un projet de création d'un serveur IRC.

## Compilation

Exécutez `make` pour compiler le projet.

## Utilisation

Exécutez `./ft_irc` pour lancer le serveur. Connectez-vous avec un client IRC.

## Ressources
